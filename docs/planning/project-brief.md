# Project Brief

## Summary

- `j-xyz.github.io` is the Eleventy source for `swiftlabs.xyz`, a small
  professional website for Jane Swift.
- The site currently presents a homepage, about page, work page, and contact
  page with shared layout and metadata from `src/_includes/layout.njk` and
  `src/_data/site.json`.

## Role In The System

- Serve as Jane Swift's public-facing home base and durable contact surface.
- Present a concise professional narrative around hospitality, product, and
  software systems.
- Keep the site easy to update so new writing or case-study pages can be added
  without reworking the whole structure.

## Near-Term Next Steps

- Keep the copy aligned with the current `site.json` description and nav
  structure.
- Update page content and metadata in `src/` before making structural
  template changes.
- Use `npm run build` after edits that affect layout, styles, or canonical
  metadata.
